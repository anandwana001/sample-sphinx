Catchflicks |logo| is a sample project after completing Android Development Online Course For Professionals by MindOrks.

.. |logo| image:: /images/app_logo.png
              :scale: 25 %

This app shows the popular movies, upcoming movies and the ongoing movies in theatre from the movie db API.

The goal of this sample project is to implement most of the learning from the Bootcamp.
Also, this project works as a kitchen sink project where anyone can play with any new Android API's and concepts.

Architecture
------------
Project consist of multiple packages in order to achieve MVVM architecture.

* data
* di
* ui
* utils
